## 设置环境变量
在终端中输入以下命令，将 your_email 和 your_password 替换为你的实际邮箱和密码：
	# Linux/macOS: 
	export SENECA_EMAIL="your_email"
	export SENECA_PASSWORD="your_password"
	# Windows: 
	set SENECA_EMAIL=your_email
	set SENECA_PASSWORD=your_password
